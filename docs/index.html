FEC Pattern Library

This is the collection of styles and design patterns that make up beta.FEC.gov. Like the main site, the style guide is a work in progress. We don’t yet have a full list of design patterns or complete documentation, but we’re actively developing these resources as we build the site.

Let us know how we can improve.
